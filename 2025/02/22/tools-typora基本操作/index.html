基本内容

目录列表

目录的生成的方法可以利用如下的语法来实现，其中的 toc 是 Table of Contents 的含义。

[toc]

可以将上面的语句放在 md 文件的任何地方，都会自动生成一份全文的目录，数量没有限制。其中生成的依据是 markdown 语法中的标题的级别。

标题

markdown 支持 6 个级别的标题。生成标题的方法有两种：
可以采用“#”的符号来开头：#+空格+正式内容
也可以直接采用快捷键
